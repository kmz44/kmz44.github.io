1. ¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Su función es estructurar el contenido de la web utilizando etiquetas.

HTML se compone de etiquetas que representan diferentes tipos de contenido, como texto, imágenes y enlaces.

2. Estructura Básica de un Documento HTML

La estructura básica de un documento HTML incluye las etiquetas necesarias para definir el contenido de una página web.

Ejemplo de Estructura Básica

<!DOCTYPE html>
<html>
  <head>
    <meta charset="UTF-8">
    <title>Mi primera página</title>
  </head>
  <body>
    <h1>Bienvenido a mi página</h1>
    <p>Este es un ejemplo de un párrafo.</p>
  </body>
</html>

3. Formato de Texto

En HTML, puedes dar formato al texto utilizando diferentes etiquetas:

  • <h1>...</h1> a <h6>...</h6> para títulos de diferentes tamaños.
  • <p>...</p> para párrafos.
  • <b>...</b> para texto en negrita.
  • <i>...</i> para texto en cursiva.

Ejemplo de Formato de Texto

<h1>Este es un título</h1>
<p>Este es un párrafo de texto.</p>
<b>Este texto está en negrita.</b>
<i>Este texto está en cursiva.</i>

4. Imágenes en HTML

Para agregar imágenes en HTML, utilizamos la etiqueta <img>. Esta etiqueta es auto-cerrada y requiere al menos el atributo src para especificar la ubicación de la imagen.

Ejemplo de una Imagen

<img src="ruta-de-la-imagen.jpg" alt="Descripción de la imagen">

9. Divs y Spans

Los elementos <div> y <span> son usados para agrupar contenido. <div> es un elemento de bloque, mientras que <span> es un elemento en línea.

10. Multimedia en HTML

HTML permite incluir elementos multimedia como videos y audio. Utilizamos la etiqueta <video> para videos y <audio> para audio.

Ejemplo de Multimedia

  <source src="video.mp4" type="video/mp4">
  Tu navegador no soporta videos.
</video>

<audio controls>
  <source src="audio.mp3" type="audio/mp3">
  Tu navegador no soporta audio.
</audio>

cómo hacer un formulario mas sofisticado necesidad de una base de datos

Para crear un formulario en HTML, utilizamos la etiqueta <form>. Dentro de esta etiqueta, podemos incluir diferentes elementos para capturar la información del usuario. A continuación, se muestra un ejemplo básico: Este código incluye varios elementos:

  • <form>: Define el inicio de un formulario. El atributo action especifica la URL a la que se enviarán los datos, y method indica el método de envío (generalmente "post").
  • <label>: Crea una etiqueta para cada campo, mejorando la accesibilidad. El atributo for asocia la etiqueta con el campo correspondiente.
  • <input>: Define un campo donde el usuario puede ingresar información. El tipo de entrada se especifica mediante el atributo type, como "text" para texto, "email" para correos, y "number" para números.
  • <select>: Crea un menú desplegable para que el usuario elija una opción. Contiene varias etiquetas <option> que representan las opciones disponibles.
  • <input type="submit">: Crea un botón que el usuario puede presionar para enviar el formulario.
Los atributos como required indican que el campo es obligatorio, y minlength asegura que el texto ingresado tenga al menos una longitud mínima.

Ejemplo de un Enlace

     
                <form action="#" method="post">

                  <label for="nombre">Nombre (mínimo 3 letras):</label>

                  <input type="text" id="nombre" name="nombre" minlength="3" required><br><br>

                  <label for="email">Email (debe contener @):</label>

                  <input type="email" id="email" name="email" required><br><br>

                  <label for="edad">Edad:</label>

                  <input type="number" id="edad" name="edad" required><br><br>

                  <label for="pais">País:</label>

                  <select id="pais" name="pais" required>

                    <option value="">Seleccione un país</option>

                    <option value="espana">España</option>

                    <option value="mexico">México</option>

                    <option value="argentina">Argentina</option>

                    <option value="colombia">Colombia</option>

                  </select><br><br>

                  <input type="submit" value="Enviar">

                </form>
